## Технические требования
1. Использовать React
2. Библиотека состояний, при необходимости: на ваш выбор
3. Библиотека UI компонентов - Consta
4. Код должен быть на TypeScript
5. Должно работать в Яндекс.Браузере и Edge
6. UI должен хорошо выглядеть на разрешениях от 1024 до 1920
7. Использовать API https://gorest.co.in/

## Функциональные требования
Главная страница
1. Поле ввода для access token. Валидация (токен не пустой).
2. Два режима: отображение постов и отображение списка пользователей. Кнопки/контроллеры/другое решение для переключения между ними.

## Список пользователей
1. Таблица с информацией:
- Имя
- Фамилия
- Email
2. Пагинация:
- Номера страниц
- Кнопки "Предыдущая"/"Следующая"
- Выбор количества элементов на странице (10/25/50)
- Клик по строке - переход на карточку пользователя

## Список постов
1. Таблица
- ID
- Заголовок
2. Аналогичная пагинация
3. Клик по строке - переход на карточку поста

## Карточка пользователя/поста
1. Для поста отображаются все комментарии
2. Возможность возврата к списку
3. Интерфейс на усмотрение разработчика

## Требования к сдаче
1. Код выложить на GitHub/GitLab
2. README.md с инструкцией по запуску
3. Деплой на любом бесплатном хостинге
