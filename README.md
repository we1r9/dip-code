# dip-code

[Веб-приложение](https://dip-code.vercel.app/) для просмотра пользователей и постов через [GoRest API](https://gorest.co.in/).

## Стек

- React + TypeScript + Vite
- React Router v6
- Consta UI ([@consta/uikit](https://consta.design/libs/uikit))

## Требования

- [Node.js](https://nodejs.org/) v18+

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone https://github.com/we1r9/dip-code.git
cd dip-code
```

2. Установите зависимости:

```bash
npm install
```

3. Запустите приложение:

```bash
npm run dev
```

## Использование

На главной странице введите Bearer-токен от [gorest.co.in](https://gorest.co.in/) и выберите раздел для просмотра.

## Что реализовано

Все функциональные требования выполнены: главная страница с валидацией токена и переключением режимов, таблицы пользователей и постов с пагинацией, карточки с возможностью возврата, комментарии к постам.

Помимо требований:

- **Обработка сетевых ошибок** – на всех страницах при потере соединения отображается заглушка с кнопкой перезагрузки.
- **Обработка 404** – если пользователь или пост удален, показывается информативная заглушка вместо пустого экрана.
- **Страница 404** для неизвестных маршрутов.
- **Токен сохраняется в sessionStorage** – не теряется при обновлении страницы.
- **Динамические заголовки вкладок** – каждая страница устанавливает свой `document.title`.
